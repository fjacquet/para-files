  <action>
Make the following targeted changes. Do NOT change any function signatures, return types, or logic outside the except blocks.

**1. Description enrichment block (lines 98-103) — replace `pass` with a warning:**

Current code:

```python
    # Try to get description (optional enrichment)
    try:
        desc = isbnlib.desc(canonical)
        if desc:
            info.description = desc
    except Exception:  # noqa: BLE001, S110
        pass  # Description is optional, fail silently
```

Replace with:

```python
    # Try to get description (optional enrichment)
    try:
        desc = isbnlib.desc(canonical)
        if desc:
            info.description = desc
    except Exception as e:  # noqa: BLE001
        logger.warning(
            "ISBN description enrichment failed for {}: {} {}",
            canonical,
            type(e).__name__,
            e,
        )
```

**2. Cover URL enrichment block (lines 106-111) — replace `pass` with a warning:**

Current code:

```python
    # Try to get cover URL (optional enrichment)
    try:
        cover = isbnlib.cover(canonical)
        if cover and "thumbnail" in cover:
            info.cover_url = cover["thumbnail"]
    except Exception:  # noqa: BLE001, S110
        pass  # Cover URL is optional, fail silently
```

Replace with:

```python
    # Try to get cover URL (optional enrichment)
    try:
        cover = isbnlib.cover(canonical)
        if cover and "thumbnail" in cover:
            info.cover_url = cover["thumbnail"]
    except Exception as e:  # noqa: BLE001
        logger.warning(
            "ISBN cover URL enrichment failed for {}: {} {}",
            canonical,
            type(e).__name__,
            e,
        )
```
