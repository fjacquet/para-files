Extension group constants (module-level dicts/sets):

```python
EXTENSION_GROUPS: dict[str, tuple[str, float]] = {
    # extension (lowercase with dot) -> (folder_attr_name, confidence)
    ".3gp":  ("media_video_folder", 0.97),
    ".m4v":  ("media_video_folder", 0.97),
    ".mp4":  ("media_video_folder", 0.97),
    ".mov":  ("media_video_folder", 0.97),
    ".m4a":  ("media_audio_folder", 0.97),
    ".mp3":  ("media_audio_folder", 0.97),
    ".gif":  ("media_images_folder", 0.97),
    ".tif":  ("media_images_folder", 0.97),
    ".tiff": ("media_images_folder", 0.97),
    ".psd":  ("media_images_folder", 0.97),
    ".p7b":  ("security_folder", 0.98),
    ".asc":  ("security_folder", 0.98),
    ".kdbx": ("security_folder", 0.98),
    ".ps1":  ("scripts_folder", 0.97),
    ".css":  ("scripts_folder", 0.97),
    ".js":   ("scripts_folder", 0.97),
    ".sh":   ("scripts_folder", 0.97),
}
```

`ExtensionRouterClassifier.__init__(self, config: ExtensionRoutingConfig) -> None`

`classify(self, content: str, metadata: FileMetadata | None = None) -> ClassificationResult | None`:
- If metadata is None, return None (no extension available).
- Normalise extension: `ext = metadata.extension.lower()` (extension already includes the dot per types.py).
- Look up ext in EXTENSION_GROUPS.
  - If found: get folder_attr and confidence. Resolve destination via `getattr(self._config, folder_attr)`.
    Return ClassificationResult with that category and Confidence(value=confidence, source=ClassificationSource.EXTENSION_ROUTER).
  - If not found AND extension is non-empty (i.e., file has some extension just not a known one):
    Return ClassificationResult with `category=self._config.catchall_folder`,
    `Confidence(value=0.80, source=ClassificationSource.EXTENSION_ROUTER)`.
  - If extension is empty string: return None (let the pipeline decide).

Add `EXTENSION_ROUTER = "extension_router"` to `ClassificationSource` in `src/para_files/types.py`.

Classifier properties:
- `name` → `"extension_router"`
- `source` → `ClassificationSource.EXTENSION_ROUTER`
- `default_confidence` → `0.97`

Use `from __future__ import annotations`. Import loguru logger. Log a DEBUG message on match.
Ensure ruff + mypy pass with no errors.
