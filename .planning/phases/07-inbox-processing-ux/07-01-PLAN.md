**`@dataclass` class `_InboxStats`:**

```python
@dataclass
class _InboxStats:
    total: int = 0
    moved: int = 0
    stayed: int = 0
    failed: int = 0
    by_signal: dict[str, int] = field(default_factory=dict)
```

**`_process_inbox_file` function:**
Signature:

```python
def _process_inbox_file(
    file_path: Path,
    pipeline: ClassificationPipeline,
    stats: _InboxStats,
    idx: int,
    total: int,
    *,
    dry_run: bool,
    conflict_strategy: ConflictStrategy,
    verbose: bool,
) -> None:
```

Logic:

1. `result = pipeline.classify_file(file_path)` — wrap in try/except Exception; on exception log warning and increment `stats.failed`, return.
2. `source = result.confidence.source`
3. If `source == ClassificationSource.DEFAULT`: increment `stats.stayed`, print `[{idx}/{total}] INBOX: {file_path.name}` (stays in inbox — not classified), return.
4. Otherwise: `target_dir = pipeline.get_target_path(result)`; call `move_classified_file(file_path, target_dir, dry_run=dry_run, copy_mode=False, conflict_strategy=conflict_strategy, add_date_prefix=False, smart_rename=False, classification=result)`.
5. If `move_result.success`: increment `stats.moved`; update `stats.by_signal[source.value]` (+1); print `[{idx}/{total}] {file_path.name}` then `-> {move_result.destination}` (right-aligned arrow).
6. If not `move_result.success`: increment `stats.failed`; print error to stderr via `typer.echo(..., err=True)`.
7. If verbose and result.signals: print signal breakdown indented (same style as move_cmd.py).

**`_print_inbox_summary` function:**
Signature:

```python
def _print_inbox_summary(stats: _InboxStats, *, dry_run: bool) -> None:
```

Logic:

- Print blank line then `--- Inbox Summary ---`
- If dry_run: prefix with `(dry run)`
- Print `Total processed : {stats.total}`
- Print `Moved           : {stats.moved}`
- Print `Stayed in Inbox : {stats.stayed}` (unclassified)
- If `stats.failed > 0`: print `Errors          : {stats.failed}`
- If `stats.by_signal`: print blank line then `By signal source:` then for each entry sorted by count desc: `{source}: {count}`

**`@app.command()` `inbox` function:**
Signature:

```python
@app.command()
def inbox(
    directory: Annotated[Path, typer.Argument(help="Inbox directory to process (default: configured para_root/0_Inbox)")] = ...,
    reference_tree: Annotated[Path | None, typer.Option("--reference-tree", "-r", help="Path to reference tree YAML file")] = None,
    dry_run: Annotated[bool, typer.Option("--dry-run/--no-dry-run", "-n", help="Preview without moving files")] = False,
    conflict: Annotated[ConflictChoice, typer.Option(help="Strategy for handling existing files")] = ConflictChoice.rename,
    verbose: Annotated[bool, typer.Option("--verbose", "-v", help="Show per-classifier signal breakdown")] = False,
) -> None:
```

Docstring: "Process all files in the inbox directory — classify and move confidently-matched files in one shot."

Logic:

1. `setup_logging(verbose=verbose)`
2. `config = load_config_or_exit()`
3. If `reference_tree`: `config.reference_tree_path = reference_tree`
4. Resolve directory: if not passed (i.e., argument not provided by user), use `config.inbox_path`. The Argument default must be `None` with `Optional[Path]` — change signature to `directory: Annotated[Path | None, typer.Argument(...)] = None`, then `dir_path = (directory or config.inbox_path).resolve()`.
5. `validate_directory_or_exit(dir_path)`
6. `files = discover_files(dir_path, recursive=False, ext_filter=None)` — non-recursive, no extension filter (process everything in inbox root).
7. If not files: `typer.echo("No files found in inbox")`, return.
8. `typer.echo(f"Processing {len(files)} file(s) from {dir_path}")` then blank line.
9. `pipeline = ClassificationPipeline(config)`
10. `conflict_strategy = ConflictStrategy(conflict.value)`
11. `stats = _InboxStats(total=len(files))`
12. Loop over `enumerate(files, 1)` — call `_process_inbox_file(f, pipeline, stats, idx, len(files), dry_run=dry_run, conflict_strategy=conflict_strategy, verbose=verbose)`.
13. `_print_inbox_summary(stats, dry_run=dry_run)`

Use `from __future__ import annotations` at top. Ensure all type hints are consistent. Do NOT use emoji in output — use plain ASCII. No bare `except:` — always catch `Exception`.
