# Photographe Amateur - Coach Personnel

Coach photo adapté à ton équipement Fuji et ton workflow timelapse.

## Équipement de référence

| Élément | Usage |
|---------|-------|
| **Fuji X-T4** | Boîtier principal (26MP, IBIS, 4K) |
| **Lightroom Classic** | Développement RAW, catalogage |
| **Unleashed** | Intervalomètre Bluetooth, contrôle à distance |
| **PhotoPills** | Planification soleil/lune, repérage |
| **LRTimelapse 6** | Workflow timelapse pro, deflicker, ramping |

## Workflow recommandé

```
Planification (PhotoPills)
    ↓
Prise de vue (X-T4 + Unleashed)
    ↓
Import & Sélection (Lightroom)
    ↓
Développement RAW (LR ou LRTimelapse)
    ↓
Export final
```

## Réglages X-T4 rapides

### Mode paysage
- Mode : **A** (priorité ouverture)
- Ouverture : **f/8-f/11** (zone de netteté optimale)
- ISO : **160** (ISO natif)
- Mesure : **Multi**
- AF : **Single Point**
- Simulation film : **Velvia** (couleurs saturées) ou **Classic Chrome**

### Mode timelapse (avec Unleashed)
- Mode : **M** (manuel complet)
- Ouverture : **f/8**
- ISO : **Auto** avec limite (pour holy grail)
- Obturateur : selon intervalle (1/100 min pour eviter flicker)
- Unleashed : intervalle 2-5s selon sujet

### Mode astro
- Mode : **M**
- Ouverture : **la plus grande possible** (f/1.4-f/2.8)
- ISO : **3200-6400**
- Mise au point : **manuelle sur étoile brillante**
- Règle 500 : temps max = 500 / focale (ex: 500/18 = 27s)

### Mode portrait
- Mode : **A** (priorité ouverture)
- Ouverture : **f/1.4-f/2.8** (flou d'arrière-plan)
- ISO : **160-800** (le plus bas possible)
- AF : **Eye AF** (détection oeil)
- Simulation film : **Astia** (tons chair doux)

## Checklist terrain

### Avant de partir
- [ ] Batteries chargées (2 minimum)
- [ ] Cartes SD formatées
- [ ] PhotoPills : vérifier heures golden/blue hour
- [ ] Trépied stable
- [ ] Téléphone chargé (Unleashed + PhotoPills)

### Sur place
- [ ] Niveau trépied
- [ ] Désactiver stabilisation (sur trépied)
- [ ] RAW uniquement
- [ ] Vérifier histogramme (pas de cramé)
- [ ] Bracketing si fort contraste

## Conseils progression

1. **Commencer simple** : maîtriser le mode A avant le mode M
2. **Un réglage à la fois** : changer une variable, observer le résultat
3. **Shooter en RAW** : toujours, pour flexibilité post-traitement
4. **Golden hour** : 1h avant/après lever/coucher = meilleure lumière
5. **Revoir ses photos** : analyser ce qui fonctionne et pourquoi
