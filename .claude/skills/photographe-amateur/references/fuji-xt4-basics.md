## Comprendre les molettes

Le X-T4 a des molettes physiques - c'est sa force pour apprendre.

```
[ISO]     [Vitesse]     [Compensation exposition]
   \          |              /
    \         |             /
     [      BOÎTIER      ]
            /   \
    [Ouverture]  [Mode AF]
     (sur objectif)
```

### Molette ISO (gauche)

- **A** : ISO automatique (recommandé pour débuter)
- **160** : ISO natif = meilleure qualité
- **160-12800** : plage utilisable sans trop de bruit

### Molette vitesse (droite)

- **A** : vitesse automatique
- **Valeurs** : 1s à 1/8000s
- Règle : vitesse min = 1/focale (ex: 1/50s pour 35mm)

### Bague ouverture (objectif)

- **A** : ouverture automatique
- **Petit nombre (f/1.4)** : flou d'arrière-plan, plus de lumière
- **Grand nombre (f/16)** : tout net, moins de lumière

## Triangle d'exposition

```
        EXPOSITION
       /    |    \
      /     |     \
   ISO -- Vitesse -- Ouverture

ISO ↑      = Plus de bruit
Vitesse ↓  = Flou de mouvement
Ouverture ↓ = Faible profondeur de champ
```

### Équivalences

Ces réglages donnent la même exposition :

- f/8, 1/125s, ISO 200
- f/5.6, 1/250s, ISO 200 (plus de flou, moins de mouvement)
- f/8, 1/250s, ISO 400 (même netteté, plus de bruit)

## Modes de prise de vue

### Mode A (Priorité ouverture) - RECOMMANDÉ POUR DÉBUTER

Tu choisis l'ouverture, l'appareil ajuste la vitesse.

- **Paysage** : f/8-f/11 pour tout net
- **Portrait** : f/2.8-f/4 pour flou d'arrière-plan
- **Usage** : 80% des situations

### Mode S (Priorité vitesse)

Tu choisis la vitesse, l'appareil ajuste l'ouverture.

- **Sport/Action** : 1/500s minimum
- **Eau soyeuse** : 1/4s à 2s
- **Usage** : sujets en mouvement

### Mode M (Manuel)

Tu contrôles tout. Indispensable pour :

- Timelapse
- Astrophoto
- Conditions difficiles
- Usage du flash

### Mode P (Programme)

L'appareil décide tout mais tu peux décaler.

- Utile pour snapshots rapides

## Menus essentiels

### Qualité d'image (IQ SETTING)

```
Menu → IQ Setting
├── Image Quality : RAW (toujours!)
├── Image Size : L (26MP)
└── Dynamic Range : Auto (DR100-400)
```

### Autofocus (AF SETTING)

```
Menu → AF/MF Setting
├── AF Mode : Single Point (précision)
├── Face/Eye Detection : ON (portraits)
├── Pre-AF : OFF (économie batterie)
└── AF Illuminator : ON (basse lumière)
```

### Configuration écran/EVF

```
Menu → Screen Setting
├── EVF Brightness : Auto
├── Preview Exposure in Manual : ON
└── Natural Live View : OFF (voir l'expo réelle)
```

### Boutons personnalisés

Suggestions de configuration :

- **Fn1** : ISO
- **Fn2** : White Balance
- **Fn3** : Film Simulation
- **Fn4** : Focus Check (loupe)
- **Rear Dial** : Compensation expo

## Simulations de film

Le X-T4 reproduit les films argentiques Fuji. À utiliser selon l'ambiance :

| Simulation         | Usage          | Caractéristiques                    |
| ------------------ | -------------- | ----------------------------------- |
| **Provia**         | Polyvalent     | Couleurs naturelles, standard       |
| **Velvia**         | Paysage        | Couleurs saturées, contrastes forts |
| **Classic Chrome** | Street, Voyage | Tons désaturés, ambiance vintage    |
| **Astia**          | Portrait       | Tons doux, teintes chair naturelles |
| **Acros**          | N&B artistique | Beaux noirs, grain fin              |
| **Eterna**         | Vidéo/Cinéma   | Faible contraste, couleurs douces   |

### Conseil

- Shooter en **RAW** permet de changer la simulation après coup dans Lightroom
- En JPEG, la simulation est définitive

## Autofocus

### Modes AF

- **AF-S** (Single) : pour sujets statiques - paysage, portrait posé
- **AF-C** (Continu) : pour sujets en mouvement - enfants, animaux
- **MF** (Manuel) : astro, macro, situations difficiles

### Zones AF

- **Single Point** : précision maximale, paysage
- **Zone** : zone élargie, sujets en mouvement prévisible
- **Wide/Tracking** : l'appareil choisit, bon avec détection visage

### Détection visage/œil

```
Menu → AF/MF Setting → Face/Eye Detection
- Face ON : détecte les visages
- Eye AF ON : focus sur l'œil le plus proche
- Right/Left Eye : forcer un œil
```

### Mise au point manuelle

1. Passer en **MF** (levier sur l'objectif ou menu)
2. Utiliser **Focus Peaking** : surligne les zones nettes
3. Utiliser **Focus Check** (loupe) pour vérifier

## IBIS (Stabilisation)

Le X-T4 a une stabilisation capteur 5 axes (IBIS).

### Quand l'activer

- **ON** : prise de vue à main levée
- **OFF** : sur trépied (obligatoire!)

### Performance

- Gain jusqu'à 6.5 stops
- Permet 1/4s à main levée en grand-angle
- Moins efficace en télé

### Réglage

```
Menu → IQ Setting → IS Mode
- Continuous : stabilisation permanente (vidéo)
- Shooting Only : stabilise au déclenchement (photo)
- OFF : sur trépied
```

## Réglages de départ recommandés

### Configuration initiale

1. **RAW uniquement** : Menu → IQ Setting → RAW
2. **ISO Auto** : Molette sur A, limite à 6400
3. **Mode A** : Molette vitesse sur A
4. **AF-S + Single Point** : pour commencer
5. **IBIS ON** : sauf sur trépied
6. **Film Simulation** : Provia (neutre)

### Premier exercice

1. Sortir en mode A, ISO Auto, f/8
2. Photographier le même sujet à f/2.8, f/8, f/16
3. Comparer la profondeur de champ
4. Observer l'histogramme

### Progression suggérée

1. **Semaine 1-2** : Mode A uniquement, comprendre l'ouverture
2. **Semaine 3-4** : Mode S pour le mouvement
3. **Semaine 5-6** : Mode M avec ISO fixe
4. **Ensuite** : Mode M complet pour timelapse/astro
