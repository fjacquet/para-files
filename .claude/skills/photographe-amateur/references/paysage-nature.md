## Composition

### Règle des tiers

Diviser l'image en 9 zones égales. Placer les éléments importants sur les lignes ou intersections.

```
+-------+-------+-------+
|       |   ×   |       |  ← Horizon sur ligne supérieure
|       |       |       |
+-------+-------+-------+
|       |       |       |
|       |       |   ×   |  ← Sujet sur intersection
+-------+-------+-------+
|       |       |       |
+-------+-------+-------+
```

**Sur X-T4** : Menu → Screen Setting → Framing Guideline → Grid 9

### Lignes directrices

Guider l'œil vers le sujet principal :

- Chemins, routes, rivières
- Clôtures, rangées d'arbres
- Ombres, lignes de crête

### Plans (profondeur)

Une bonne photo de paysage a 3 plans :

1. **Premier plan** : élément proche (rocher, fleur, vague)
2. **Plan moyen** : sujet principal
3. **Arrière-plan** : contexte (montagne, ciel)

### Cadrage

| Type           | Usage                      |
| -------------- | -------------------------- |
| **Horizontal** | Paysages larges, panoramas |
| **Vertical**   | Cascades, forêts, reflets  |
| **Carré**      | Minimalisme, symétrie      |

### Erreurs courantes

- Horizon penché (utiliser niveau)
- Horizon au centre (choisir ciel OU sol)
- Pas de premier plan (image plate)
- Trop d'éléments (simplifier)

## Lumière naturelle

### Golden Hour

**Quand** : 1h après lever / 1h avant coucher du soleil
**Caractéristiques** : Lumière chaude, ombres longues, contraste doux
**Usage** : Paysages, portraits extérieurs

### Blue Hour

**Quand** : 20-40 min avant lever / après coucher
**Caractéristiques** : Lumière bleue uniforme, pas d'ombres
**Usage** : Villes illuminées, ambiances mystérieuses

### Heure magique (Magic Hour)

**Quand** : Juste avant/après golden hour
**Caractéristiques** : Ciel coloré, lumière dramatique
**Usage** : Ciels spectaculaires, silhouettes

### Milieu de journée

**Problèmes** : Lumière dure, ombres courtes, contraste excessif
**Solutions** :

- Chercher l'ombre
- Photographier vers le haut (forêts)
- Noir et blanc
- Revenir à golden hour

### Contre-jour

**Avantages** : Silhouettes, rayons, atmosphère
**Technique** :

1. Mesurer sur le ciel
2. Sous-exposer de 1-2 stops
3. Utiliser pare-soleil
4. Bracketing pour HDR

### Temps couvert

**Avantages** : Lumière douce, couleurs saturées
**Idéal pour** : Cascades, forêts, macro, portraits

## Réglages terrain

### Configuration standard paysage

```
Mode        : A (priorité ouverture)
Ouverture   : f/8-f/11 (sweet spot)
ISO         : 160 (base)
Mesure      : Multi
AF          : AF-S, Single Point
Stabilisation: OFF (sur trépied)
Film Sim    : Velvia ou Provia
Format      : RAW
```

### Hyperfocale (tout net)

Distance de mise au point où tout est net de la moitié de cette distance à l'infini.

| Focale | f/8   | f/11  | f/16 |
| ------ | ----- | ----- | ---- |
| 16mm   | 1.3m  | 0.9m  | 0.6m |
| 23mm   | 2.7m  | 1.9m  | 1.3m |
| 35mm   | 6.1m  | 4.4m  | 3.0m |
| 56mm   | 15.7m | 11.4m | 7.8m |

**Astuce** : PhotoPills a un calculateur d'hyperfocale intégré

### Mise au point paysage

1. **Méthode 1/3** : Faire le point au 1/3 de la scène
2. **Hyperfocale** : Utiliser le tableau ci-dessus
3. **Focus stacking** : Plusieurs photos à différentes distances (avancé)

## Techniques avancées

### Bracketing d'exposition (HDR)

Pour scènes à fort contraste (contre-jour, intérieur/extérieur).

**Sur X-T4** :

```
Menu → Shooting Setting → AE Bracketing
- Frames : 3 ou 5
- Steps : 1 EV ou 2 EV
```

**Workflow** :

1. Trépied obligatoire
2. Bracketing 3 images (-2, 0, +2 EV)
3. Fusion dans Lightroom (Photo → Fusion HDR)

### Pose longue

Effets : eau soyeuse, nuages filés, foules disparues.

**Réglages** :

```
Mode : M
Ouverture : f/11-f/16
ISO : 160
Vitesse : 1s à 30s (selon effet)
Filtre ND : selon luminosité
```

**Temps indicatifs** :
| Effet | Temps |
|-------|-------|
| Eau légèrement floue | 1/4s - 1s |
| Eau soyeuse | 2s - 10s |
| Nuages filés | 30s - 2min |
| Effet fantôme | 10s - 30s |

### Panorama

**Technique** :

1. Trépied + tête panoramique (idéal)
2. Mode M, réglages fixes
3. Chevauchement 30-40%
4. Rotation autour du point nodal
5. Fusion dans Lightroom (Photo → Fusion Panorama)

**Erreurs** :

- Balance des blancs auto (variations)
- Sujets en mouvement dans la scène
- Pas assez de chevauchement

## Filtres

### Filtre polarisant (CPL)

**Effets** :

- Élimine reflets (eau, vitres)
- Sature les couleurs
- Assombrit le ciel bleu
- Perce la brume

**Usage** :

- Tourner jusqu'à effet désiré
- Maximum à 90° du soleil
- Éviter sur ultra grand-angle (vignettage)
- Perte 1-2 stops de lumière

### Filtres ND (Neutral Density)

Réduisent la lumière pour permettre poses longues.

| Filtre            | Réduction | Usage                         |
| ----------------- | --------- | ----------------------------- |
| ND8 (3 stops)     | ÷8        | Eau en journée                |
| ND64 (6 stops)    | ÷64       | Nuages, cascades              |
| ND1000 (10 stops) | ÷1024     | Plein soleil, effets extrêmes |

**Calcul temps de pose avec ND** :
Temps avec ND = Temps sans ND × facteur ND

Exemple : 1/125s sans filtre → 1/125 × 1000 = 8s avec ND1000

### Filtres GND (Gradués)

Équilibrent ciel lumineux et sol sombre.

- **Hard GND** : horizon net (mer, plaine)
- **Soft GND** : horizon irrégulier (montagnes)
- **Reverse GND** : couchers de soleil

## Situations courantes

### Montagne

- **Heure** : Lever/coucher pour lumière rasante
- **Composition** : Premier plan (fleurs, rochers), sommets en arrière-plan
- **Réglages** : f/11, ISO 160, exposer pour les crêtes
- **Attention** : UV intense = voile, utiliser polarisant

### Cascade

- **Heure** : Temps couvert ou ombre (éviter soleil direct)
- **Réglages** : f/11, ISO 160, 1/4s à 2s
- **Matériel** : Trépied, ND si trop lumineux
- **Astuce** : Inclure rochers mouillés = reflets

### Forêt

- **Heure** : Matin brumeux idéal
- **Composition** : Rayons de soleil, troncs comme lignes
- **Réglages** : f/8, regarder vers le haut
- **Attention** : Mesure spot sur zone moyenne

### Lac / Reflets

- **Heure** : Aube (eau calme)
- **Composition** : Symétrie, horizon au milieu acceptable
- **Réglages** : f/8, polarisant à 45° pour garder certains reflets
- **Astuce** : Attendre absence de vent

### Coucher de soleil

- **Technique** : Arriver 1h avant, rester 30min après
- **Composition** : Silhouettes, reflets, nuages
- **Réglages** : Mesure sur ciel, bracketing
- **Attention** : Ne pas regarder le soleil dans l'EVF

### Neige

- **Problème** : Sous-exposition (l'appareil grise la neige)
- **Solution** : Surexposer +1 à +2 EV
- **Balance blanc** : Ombre ou Nuageux (évite bleu)
- **Astuce** : Histogramme vers la droite
