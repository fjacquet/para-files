## Fondamentaux

### Triangle du portrait réussi
```
     Lumière
        △
       / \
      /   \
     /     \
Expression ─── Composition
```

### Distance focale et perspective

| Focale | Effet | Usage |
|--------|-------|-------|
| 23mm | Distorsion, nez agrandi | Environnement, créatif |
| 35mm | Légère distorsion | Portrait environnemental |
| 50mm (33mm APS-C) | Naturel | Polyvalent |
| **56mm** | **Flatteur, compression** | **Portrait idéal** |
| 85mm+ (56mm+ APS-C) | Très flatteur | Headshot, beauté |

### Hauteur de l'appareil
- **Au niveau des yeux** : Neutre, naturel
- **Légèrement au-dessus** : Amincit visage, grands yeux
- **En dessous** : Puissant, dominant (rare)

## Éclairage portrait

### Lumière naturelle

**Fenêtre (le plus simple)** :
```
Position modèle : 45° de la fenêtre
Distance : 1-2m
Heure : Éviter soleil direct (voilage si besoin)
```

**Extérieur** :
- **Ombre ouverte** : Sous arbre, porche, bâtiment
- **Golden hour** : Lumière chaude, flatteuse
- **Temps couvert** : Softbox géante naturelle

**Réflecteur** :
- Blanc : Doux, naturel
- Argent : Plus punch, yeux brillants
- Or : Réchauffant

### Schémas d'éclairage classiques

**Rembrandt** (dramatique) :
```
    💡
     \  45°
      \
       😊
         \
          📷
```
- Triangle de lumière sur joue ombragée
- Nez fait ombre vers joue

**Loop** (polyvalent) :
```
    💡
     | 30-45°
     |
    😊
     |
    📷
```
- Ombre du nez vers commissure
- Le plus courant

**Split** (dramatique) :
```
💡 ─── 😊 ─── 📷
         90°
```
- Moitié visage éclairée
- Moitié dans l'ombre

**Butterfly/Paramount** (beauté) :
```
       💡
        |
       😊
        |
       📷
```
- Lumière au-dessus, face
- Ombre papillon sous le nez

**Clamshell** (beauté, mode) :
```
       💡 haute
        |
       😊
        |
       ⬜ réflecteur
        |
       📷
```
- Lumière très douce
- Catchlights doubles

### Catchlights (reflets yeux)
- Rendent les yeux vivants
- Position idéale : 10h ou 2h
- Forme révèle le type de lumière

## Composition et cadrage

### Types de cadrage

| Cadrage | Coupe | Usage |
|---------|-------|-------|
| **Plan large** | Corps entier | Mode, contexte |
| **Plan américain** | Mi-cuisse | Polyvalent |
| **Plan moyen** | Taille | Portrait classique |
| **Plan rapproché** | Poitrine | Portrait standard |
| **Gros plan** | Visage | Émotion, beauté |
| **Très gros plan** | Détail (yeux) | Artistique |

### Règles de composition

**Règle des tiers** :
- Yeux sur ligne supérieure
- Visage dans un tiers

**Espace regard** :
```
Bon :          Mauvais :
┌─────────┐    ┌─────────┐
│    →😊  │    │  😊→    │
│         │    │         │
└─────────┘    └─────────┘
```
- Espace devant le regard, pas derrière

**Lignes de coupe à éviter** :
- Cou
- Articulations (coudes, genoux, poignets)
- Couper au milieu du front

### Arrière-plan
- **Simple** : Moins distrayant
- **Bokeh** : f/1.4-f/2.8 pour flou
- **Contexte** : Si apporte à l'histoire
- **Vérifier** : Poteaux, branches "sortant" de la tête

## Direction du modèle

### Mettre à l'aise
- Conversation détendue
- Montrer les images (confiance)
- Complimenter sincèrement
- Pauses régulières

### Instructions clés

**Posture** :
- "Redresse-toi, épaules en arrière"
- "Poids sur la jambe arrière"
- "Penche légèrement vers moi"

**Visage** :
- "Menton légèrement en avant et vers le bas"
- "Tourne la tête doucement vers la lumière"
- "Regarde juste au-dessus de l'objectif"

**Yeux** :
- "Pense à quelque chose de drôle" (sourire naturel)
- "Regarde au loin, puis vers moi au signal"
- "Ferme les yeux, ouvre au 3"

**Mains** :
- Toujours occupées (poche, visage, objet)
- Doigts détendus, pas crispés
- Éviter paumes face caméra

### Expressions naturelles
```
Techniques :
- Raconter blagues
- Poser questions personnelles
- Faire bouger entre les poses
- Musique
- "Fais comme si..."
```

## Réglages X-T4

### Configuration portrait
```
Mode        : A (priorité ouverture)
Ouverture   : f/1.4-f/2.8 (flou fond)
             f/4-f/5.6 (groupes, contexte)
ISO         : 160-800 (le plus bas possible)
Mesure      : Spot sur visage
AF          : Eye AF (détection oeil)
Film Sim    : Astia (tons chair doux)
             Classic Chrome (style)
```

### Eye AF (autofocus oeil)
```
Menu → AF/MF Setting → Face/Eye Detection
- Face Detection : ON
- Eye Detection : ON
- Right/Left Eye Priority : Auto ou selon composition
```

**Astuce** : AF-C + Eye AF pour modèle en mouvement

### Burst mode
Pour expressions naturelles :
```
Drive : CH (haute vitesse)
Mitrailler pendant action/rire
Sélectionner meilleure expression après
```

### Balance des blancs
- **Auto** : Généralement bon
- **Kelvin manuel** : 5500K extérieur jour, ajuster selon ambiance
- **Plus chaud** : Portraits plus flatteurs

## Situations courantes

### Portrait extérieur - Soleil
**Problème** : Ombres dures, yeux plissés

**Solutions** :
1. **Ombre ouverte** : Sous porche, arbre, bâtiment
2. **Contre-jour** : Soleil derrière, réflecteur devant
3. **Golden hour** : Éviter midi
4. **Diffuseur** : Entre soleil et sujet

### Portrait extérieur - Couvert
Conditions idéales !
- Lumière douce naturelle
- Pas d'ombres dures
- Couleurs saturées

### Portrait intérieur - Fenêtre
```
1. Placer sujet à 45° de la fenêtre
2. Distance : 1-2m de la fenêtre
3. Réflecteur côté ombre (optionnel)
4. Voilage si soleil direct
```

### Portrait de groupe
```
Réglages :
- f/5.6 - f/8 (tout le monde net)
- Aligner têtes sur même plan focal
- Varier hauteurs (assis/debout)
- Flash ou réflecteur pour uniformiser
```

### Portrait enfant
- **Se mettre à leur niveau** (physiquement)
- **Mode rafale** : Expressions spontanées
- **Jeux** : Bulles, jouets, sons
- **Patience** : Ne pas forcer

### Portrait avec lunettes
- Incliner légèrement les lunettes
- Lumière plus haute (évite reflets)
- Angle appareil légèrement décalé
- Post : Clone stamp si reflet

## Post-traitement portrait

### Lightroom - Workflow de base

#### 1. Exposition et tons
```
Profil : Camera Astia ou Portrait
Exposure : Peau bien exposée
Highlights : -30 (récupérer hautes lumières)
Shadows : +20 (détails ombres)
Whites/Blacks : Ajuster pour contraste
```

#### 2. Tons chair (HSL)
```
Orange :
  Saturation : -15 (moins orange)
  Luminance : +15 (peau lumineuse)

Rouge :
  Saturation : -20 (réduit rougeurs)
  Luminance : +10

Jaune :
  Saturation : -10 (teint uniforme)
```

#### 3. Yeux (pinceau local)
```
Iris :
  Exposure : +0.3
  Clarity : +15
  Saturation : +10

Blanc des yeux :
  Exposure : +0.2
  Saturation : -20
```

#### 4. Peau (pinceau local)
```
Adoucir :
  Clarity : -25
  Texture : -20
  Sharpness : -20

Masque : Peindre sur peau
Éviter : Yeux, sourcils, lèvres, cheveux
```

#### 5. Lèvres (optionnel)
```
Saturation : +15
Luminance : +5
```

### Retouche avancée

**Dodge & Burn** :
- Éclaircir : Front, pommettes, nez, menton
- Assombrir : Contours visage, sous menton

**Correction imperfections** :
- Spot Removal (Q) : Boutons, taches
- Toujours en mode "Heal"

### Presets portrait suggérés

| Preset | Style |
|--------|-------|
| Clean Beauty | Peau douce, yeux vifs, naturel |
| Moody Portrait | Désaturé, ombres profondes |
| Film Look | Grain, fade, tons vintage |
| High Fashion | Contraste, netteté, dramatique |
