## Approche et éthique

### Principes fondamentaux

- **Respect absolu** du modèle
- **Communication claire** avant, pendant, après
- **Consentement explicite** pour chaque pose/usage
- **Environnement sécurisant** (température, intimité)

### Contrat modèle (Model Release)

Indispensable, même entre amis. Doit préciser :

- Usage des images (portfolio, expo, vente, web)
- Durée des droits
- Rémunération éventuelle
- Droit de retrait

### Pendant le shooting

- Toujours demander avant de toucher/ajuster
- Proposer un accompagnant si le modèle le souhaite
- Pauses régulières
- Montrer les images au fur et à mesure (confiance)

## Préparation du shooting

### Discussion préalable

```
Sujets à aborder :
- Niveau de nudité souhaité (impliqué, partiel, intégral)
- Zones à ne pas montrer
- Style recherché (classique, moderne, sombre, lumineux)
- Références visuelles (moodboard)
- Utilisation finale des images
```

### Checklist lieu

- [ ] Température confortable (22-24°C minimum)
- [ ] Intimité garantie (rideaux, verrous)
- [ ] Peignoir/serviette à disposition
- [ ] Musique d'ambiance (optionnel)
- [ ] Eau et snacks

### Matériel suggéré

- Objectif portrait : 56mm f/1.2 ou 35mm f/1.4
- Réflecteur pliable (blanc/argent)
- Draps/tissus neutres (noir, blanc, beige)
- Trépied (poses longues, low-key)

## Éclairage

### Lumière naturelle

Le plus accessible et flatteur pour débuter.

**Fenêtre (soft light)** :

```
┌─────────────────────┐
│                     │
│  ▓▓▓   ←Fenêtre     │
│  ▓▓▓                │
│         🧍 Modèle   │
│                     │
│      📷            │
└─────────────────────┘
```

- Modèle à 1-2m de la fenêtre
- Lumière latérale = modelé du corps
- Voilage = adoucit les ombres

**Golden hour intérieur** :

- Lumière chaude, douce
- Ombres allongées flatteuses

### Éclairage studio simplifié

**Un seul flash/LED (Rembrandt)** :

```
       💡 Flash à 45°
        \
         \
          🧍 Modèle

      📷
```

- Crée triangle de lumière sur joue opposée
- Dramatique, sculptural

**Clamshell (beauté)** :

```
      💡 Lumière haute
       |
      🧍 Modèle
       |
      ⬜ Réflecteur bas

      📷
```

- Lumière douce, flatteuse
- Peu d'ombres

### Low-key vs High-key

| Style        | Fond  | Lumière                   | Ambiance               |
| ------------ | ----- | ------------------------- | ---------------------- |
| **Low-key**  | Noir  | Directionnelle, contraste | Dramatique, sculptural |
| **High-key** | Blanc | Diffuse, uniforme         | Doux, éthéré           |

## Composition et poses

### Règles de base

- **Lignes du corps** : Courbes naturelles, éviter rigidité
- **Angles flatteurs** : Légèrement en plongée ou contre-plongée douce
- **Espaces négatifs** : Le vide fait partie de l'image
- **Cadrage serré** : Fragments (dos, épaule, main) souvent plus forts

### Poses classiques

**Allongée** :

- Sur le côté, jambe pliée
- Sur le dos, bras au-dessus de la tête
- À plat ventre, tête tournée

**Assise** :

- Genoux ramenés, bras autour
- En tailleur, dos courbé
- Sur tabouret, profil

**Debout** :

- Poids sur une jambe (déhanché naturel)
- Bras qui cassent la symétrie
- Dos tourné, regard par-dessus l'épaule

### Direction du modèle

```
Verbes utiles :
- "Allonge ton cou"
- "Relâche les épaules"
- "Tourne légèrement vers la lumière"
- "Imagine que tu..."
- "Garde cette position, respire"
```

### Éviter

- Mains crispées (doigts détendus)
- Double menton (menton légèrement en avant)
- Bras collés au corps (créer espace)
- Regard fixe vide (donner une intention)

## Réglages X-T4

### Configuration portrait/nu

```
Mode        : A (priorité ouverture)
Ouverture   : f/2 - f/4 (flou d'arrière-plan)
ISO         : Le plus bas possible (160-400)
Mesure      : Spot (sur peau moyenne)
AF          : Eye AF ou Single Point
Film Sim    : Astia (tons chair doux) ou Classic Chrome
```

### Gestion des tons chair

- **Exposer pour la peau** : Ni cramée, ni sous-exposée
- **Histogramme** : Pic principal au centre-droit
- **Astuce** : Légère surexposition (+0.3 à +0.7) pour peau lumineuse

### Objectifs recommandés

| Objectif          | Usage                   | Rendu                   |
| ----------------- | ----------------------- | ----------------------- |
| **XF 56mm f/1.2** | Portrait serré, détails | Bokeh crémeux, flatteur |
| **XF 35mm f/1.4** | Corps entier, contexte  | Polyvalent              |
| **XF 23mm f/2**   | Environnement, créatif  | Léger grand-angle       |
| **XF 90mm f/2**   | Distance, compression   | Très flatteur           |

### Mise au point

- **Eye AF** : Pour portraits serrés
- **Single Point** : Sur l'oeil le plus proche ou zone clé
- **Manuel** : Pour poses statiques, focus précis

## Post-traitement

### Lightroom - Workflow peau

#### 1. Tons de base

```
Profil : Camera Astia ou Portrait
Exposition : Ajuster pour peau lumineuse
Highlights : -20 à -40 (récupérer hautes lumières peau)
Shadows : +10 à +20 (détails ombres douces)
```

#### 2. Couleur peau (HSL)

```
Orange :
  Hue : 0 à +5 (évite jaune maladif)
  Saturation : -10 à -20 (moins orange)
  Luminance : +10 à +20 (peau lumineuse)

Rouge :
  Saturation : -10 (réduit rougeurs)
```

#### 3. Retouche locale (pinceau)

```
Adoucir peau :
- Clarity : -20 à -40
- Texture : -20 à -30
- Peindre sur peau (éviter yeux, lèvres, cheveux)

Dodge & Burn :
- Exposure +0.3 : Éclaircir zones clés
- Exposure -0.3 : Accentuer ombres/courbes
```

#### 4. Netteté

```
Amount : 30-50 (modéré)
Masking : 70-90 (évite sharpening sur peau)
```

### Retouche avancée (Photoshop)

- **Frequency Separation** : Texture vs couleur
- **Dodge & Burn** : Sculpter lumière
- **Liquify** : Ajustements subtils (controversé)

### Styles de post-traitement

| Style         | Caractéristiques                      |
| ------------- | ------------------------------------- |
| **Classique** | Tons naturels, contraste modéré       |
| **Fine Art**  | N&B, grain, contraste élevé           |
| **Lumineux**  | High-key, tons pastels, aérien        |
| **Sombre**    | Low-key, ombres profondes, dramatique |
| **Film**      | Grain, couleurs désaturées, vintage   |

## Inspirations et styles

### Photographes de référence

- **Edward Weston** : Formes, abstraction
- **Helmut Newton** : Glamour, puissant
- **Ruth Bernhard** : Lumière sculptée
- **Herb Ritts** : Graphique, contrasté
- **Paolo Roversi** : Éthéré, romantique

### Exercices pour progresser

**1. Silhouettes**

- Contre-jour fenêtre
- Exposer pour le fond
- Forme pure du corps

**2. Fragments**

- Cadrer serré (main, épaule, dos)
- Abstraction
- Lumière latérale forte

**3. Clair-obscur**

- Une seule source latérale
- Fond noir
- Inspiration peinture baroque

**4. Autoportrait / Mannequin**

- Pratiquer éclairage sans modèle
- Tester angles et cadrages
- Aucune pression temps

- **Flickr** : Avec filtre adulte
- **Portfolio personnel** : Contrôle total
- **Expositions locales** : Galeries, festivals photo
