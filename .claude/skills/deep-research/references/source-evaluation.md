# Évaluation des Sources

## Critères CRAAP

Méthode académique standard pour évaluer la qualité d'une source.

### C - Currency (Actualité)

| Question | Critère |
|----------|---------|
| Date de publication ? | Récente = mieux (sauf historique) |
| Dernière mise à jour ? | Contenus maintenus = fiables |
| Liens fonctionnels ? | Liens cassés = source abandonnée |
| Sujet sensible au temps ? | Sciences, tech = besoin de récence |

**Score** : /5

### R - Relevance (Pertinence)

| Question | Critère |
|----------|---------|
| Répond à ma question ? | Directement pertinent |
| Public cible ? | Académique > grand public |
| Niveau approprié ? | Ni trop basique, ni trop spécialisé |
| Meilleures sources disponibles ? | Comparer avec alternatives |

**Score** : /5

### A - Authority (Autorité)

| Question | Critère |
|----------|---------|
| Auteur identifié ? | Anonyme = méfiance |
| Qualifications ? | Expert du domaine |
| Affiliations ? | Université, institution reconnue |
| Contact disponible ? | Transparence |

**Score** : /5

### A - Accuracy (Exactitude)

| Question | Critère |
|----------|---------|
| Sources citées ? | Références vérifiables |
| Preuves fournies ? | Données, études, documents |
| Vérifiable ailleurs ? | Triangulation possible |
| Revu par des pairs ? | Peer-review = haute qualité |
| Erreurs factuelles ? | Fautes = méfiance |

**Score** : /5

### P - Purpose (Objectif)

| Question | Critère |
|----------|---------|
| But de l'information ? | Informer, persuader, vendre ? |
| Objectif vs opinion ? | Faits séparés des avis |
| Biais évidents ? | Politique, commercial, idéologique |
| Ton émotionnel ? | Sensationnalisme = méfiance |

**Score** : /5

## Grille de Notation

| Score Total | Fiabilité | Usage |
|-------------|-----------|-------|
| 20-25 | Excellente | Source primaire, citer directement |
| 15-19 | Bonne | Utiliser avec d'autres sources |
| 10-14 | Moyenne | Vérifier avec sources meilleures |
| 5-9 | Faible | Éviter ou mentionner les réserves |
| 0-4 | Très faible | Ne pas utiliser |

## Types de Sources et Fiabilité

### Très Haute Fiabilité
- Publications académiques peer-reviewed
- Rapports d'organisations internationales (OMS, ONU)
- Données gouvernementales officielles
- Méta-analyses et revues systématiques

### Haute Fiabilité
- Livres d'universitaires reconnus
- Articles de revues spécialisées
- Rapports d'instituts de recherche
- Journaux de référence (fact-checkés)

### Fiabilité Moyenne
- Articles de presse de qualité
- Blogs d'experts reconnus
- Encyclopédies (Wikipedia avec sources)
- Présentations de conférences

### Fiabilité Variable
- Réseaux sociaux
- Blogs personnels
- Forums et Q&A
- Communiqués de presse

### À Éviter
- Sources anonymes non vérifiables
- Sites conspirationnistes
- Contenus sponsorisés non identifiés
- Informations sans date ni auteur

## Signaux d'Alerte (Red Flags)

### Dans le Contenu
- Absence de sources/références
- Généralisations excessives ("Tout le monde sait que...")
- Langage émotionnel ou sensationnaliste
- Affirmations extraordinaires sans preuves
- Contradictions internes

### Dans la Source
- Auteur anonyme ou pseudonyme
- Pas de date de publication
- Domaine suspect (.xyz, .info parfois)
- Design amateur ou publicités excessives
- Pas de mentions légales

### Dans la Diffusion
- Partagé uniquement sur réseaux sociaux
- Titre clickbait
- Urgence artificielle ("Avant censure !")
- Appel aux émotions fortes

## Checklist Rapide

```
[ ] Auteur identifié et qualifié ?
[ ] Date récente ou pertinente ?
[ ] Sources citées et vérifiables ?
[ ] Confirmé par 2+ autres sources ?
[ ] Pas de biais commercial/politique évident ?
[ ] Pas de signaux d'alerte ?
```

Si 6/6 : Source utilisable
Si 4-5/6 : Utiliser avec précautions
Si <4/6 : Chercher une meilleure source
